1.什么是函数
函数是这样一段代码，定义一次，调用多次
2.函数的语法
function funName(参数0，多个...){
	//执行代码
	// return 参数;

}

3.函数的定义
函数的声明

function funName(参数0，多个...){
	//执行代码
	// return 参数;

}


函数的表达式

var funName = function(参数0，多个...){
	//执行代码
	// return 参数;
}

区别：
对于函数的声明，函数的调用可以在任何地方
对于函数的表达式，必须放在函数的后面

4.函数的调用
funName(实参0，多个);

当实参多于形参，多于的实参无效
当形参多于实参，多于的形参用undefined填充

5.作用域
全局作用域

1.最外层函数和最外层函数的外面声明的变量都拥有全局作用域
2.所有未声明直接使用的变量自动拥有全局作用域
3.window对像上的所有属性和方法都拥有全局作用域


局部作用域
除了全局作用域之外的其他变量和方法都是局部作用域


访问范围：
全局不可以访问局部
但是局部可以访问全局
